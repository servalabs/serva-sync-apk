# Serva Sync

Serva Sync is a secure and decentralized file synchronization app that allows you to sync files across your devices without relying on cloud storage.

It enables direct device-to-device file transfer with privacy, speed, and full control over your data.

## 📥 Download

Get the latest APK from the Releases section:
https://github.com/servalabs/serva-sync-apk/releases

## 📱 Install via Serva Store

You can install and receive updates automatically using Serva Store:

1. Open Serva Store
2. Add App
3. Enter this repository URL:
   https://github.com/servalabs/serva-sync-apk
4. Install
